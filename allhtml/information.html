- What is Phoenix SRV?

Phoenix SRV is a Minecraft server created on 06/12/2021 by @Adrillox919 and @Ewan1011. Phoenix SRV is a totally free open world Minecraft server, in which you can enjoy this magnificent experience thanks to the missions that can be found, and the mods that the server has.

- How can I join the server?

In order to join the Phoenix SRV Discord server you have to go back to the main page (by clicking the Home button on the menu) and you have to scroll until you find the section called "Our Discord". Once you are in this section you have to click on the button below and it will redirect you to our Discord server.

If the button or link does not work, contact the STAFF in the "Contact us" section.

- Where can I find the IP?

The IP of the minecraft Phoenix SRV server is exclusively found on the Discord server. In the Discord server you have to first read the manual and the instructions and be able to pass the manual verification to be able to unblock the channel where the IP is located. The reason why we do not add the IP on the website is to avoid any attacks to the server.

- Do I need to be premium?

Phoenix SRV is a completely free Minecraft server open to the public, so you don't need to be premium. The only 2 requirements necessary to enter the server are the following:

1: Have read the manual and instructions.

2: Ask to join the whitelist
